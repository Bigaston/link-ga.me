LinkGame

Create an easy to share link for your games!

LinkGames is a free website to share your games and all the place where people can find it!
All the information about your game are store in the URL, and the basic source code is OpenSource (you can find the code on Github).

How use LinkGame

1- Go to link-ga.me

2- Complete the information you have. You can use some image host system or use image on your website

3- Now you can get the URL and share it (or use some URL Shortner)
